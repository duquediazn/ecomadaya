# ADR (Architecture Decision Records)

Los ADR registran decisiones tecnicas importantes del proyecto.

## Convencion

- Un archivo por decision: `NNNN-titulo-corto.md`
- NNNN correlativo: 0001, 0002, 0003...

## Estado posible

- Proposed
- Accepted
- Superseded
- Rejected
